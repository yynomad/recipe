# 上班族自动减脂食谱生成工具
专门适配16+8减脂法的自动食谱生成工具，每天定时推送第二天的早午饭，不用自己想每天吃什么，减腹部脂肪，完美适配下班晚的上班族作息。
## ✨ 功能特点
1. 🕒 自动定时推送：默认每天18:00推送第二天的早午饭，不用手动查询
2. 🎯 专属定制：针对上班族需求，早餐快手5分钟搞定，午餐可带便当/点外卖
3. 🚫 自动去重：自动记录历史食谱，保证至少3个月不重样
4. 🍱 减脂友好：低油低糖高蛋白高纤维，专门针对减腹部脂肪设计
## 🚀 快速使用
### 1. 安装依赖
```bash
pip3 install pyyaml requests apscheduler
```
### 2. 修改配置
打开`config.yaml`，把`doubao_api_key`换成你自己的豆包API密钥，可以自定义推送时间、饮食偏好。
### 3. 运行脚本
```bash
# 前台运行测试
python3 auto_menu.py
# 后台常驻运行
nohup python3 auto_menu.py > menu.log 2>&1 &
```
### 4. 开机自启（可选）
把启动命令加到系统开机启动项里，开机自动运行，不用每次手动启动。
## 📁 文件说明
| 文件名 | 说明 |
| --- | --- |
| `auto_menu.py` | 主脚本，自动生成+推送食谱 |
| `config.yaml` | 配置文件，修改API密钥、推送时间、饮食偏好 |
| `menu_history.json` | 自动生成的历史记录文件，用来去重 |
## 💡 自定义配置
可以在`config.yaml`的`diet_preference`里加自己的饮食偏好，比如：
- 不吃香菜、喜欢吃海鲜、不爱吃猪肉
- 要求食材都是常见的，不要小众食材
- 要求做法更简单，尽量不用开火
- 适配素食/健身增肌等特殊需求
## 🔧 部署说明
可以部署在本地电脑、云服务器、家里的NAS上，只要能联网就能一直运行，每天自动推送。
